# Game Development Projects

This repository contains a collection of game projects developed using **Unity** and **C#**.
Each folder represents a different game or game assignment created to learn and practice game design, programming, and 3D graphics.

The projects focus on gameplay mechanics, physics, user interaction, and visual effects.

---

## What You Will Find Here

* 2D and 3D game projects
* Player movement and controls
* Physics and collision systems
* Animations and particle effects
* Game UI and scoring systems
* Unity scenes and scripts written in C#

---

## Project Folders

* **Assignment 1** – Basic game mechanics and player control
* **Assignment 2** – Physics, collisions, and environment interaction
* **Assignment 3** – Advanced gameplay logic and visual effects

Each folder contains:

* Unity project files
* C# scripts
* Scenes and assets
* Prefabs and materials

---

## How to Run a Project

1. Install **Unity Hub**
2. Click **Open Project**
3. Select one of the assignment folders
4. Open the main scene from the `Scenes` folder
5. Click the **Play** button in Unity

* Unity Engine
